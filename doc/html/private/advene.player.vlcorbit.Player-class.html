Wrapper class for a VLC.MediaControl object.

It provides some helper methods, and forwards other requests to the VLC.MediaControl object.

update_status(self, status=None, position=None)

Update the player status.

Defined status:
  • start
  • pause
  • resume
  • stop
  • set

If no status is given, it only updates the value of self.status

If position is None, it will be considered as zero for the "start" action, and as the current relative position for other actions.
Parameters:
status - the new status
           (type=string)
position - the position
           (type=VLC.Position)
