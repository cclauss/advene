Specialized HTTP server for the Advene framework.

This is a specialized HTTP Server dedicated to serving Advene packages content, and interacting with a media player.

__init__(self, controller=None, port=1234)
(Constructor)

HTTP Server initialization.

When running embedded, the server should be provided a controller parameter, responsible for handling package loading and player interaction. When running standalone, the server will its own controller.
Parameters:
controller - the controller
           (type=any, should provide some methods (load_package, update_status, ...))
port - the port number the server should bind to
           (type=int)
Overrides:

verify_request(self, request, client_address)

Access control method.

This method returns True if the client is allowed to send this request. We use the authorized_hosts list to check.
Parameters:

           (type=Request)
