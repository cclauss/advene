Type _match_filter_dict

object --+    
         |    
      dict --+

            _match_filter_dict

A specific dictionnary meant to reflect the state of both attributes 'viewable-class' and 'viewable-type' of a 'view' element. The corresponding keys are 'class' and 'type', respectively.

It also has a _class_ and a _type_ property.
