getQName(self, key, namespaces, default=None)

Resolve the given key as a QName with regard to the given namespaces.

_namespaces_ is a dict whose keys are qname prefices and values are URIs. QName is resolved by concatenating the URI, '#' and the suffix.

keys(self)

Return the keys of this bundle.

Note that items in the bundle are indexed by their keys. Hence, this method is equivalent to _uris_.
