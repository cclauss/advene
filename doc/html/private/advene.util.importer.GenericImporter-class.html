Method Details

convert(self, source)

Converts the source elements to annotations.

Source is an iterator or a list returning dictionaries. The following keys MUST be defined:
  • begin (in ms)
  • end or duration (in ms)
  • content
The following keys are optional:
  • id
  • type (which must be a *type*, not a type-id)
