getUri(self, absolute=True)

Return the content's URI.

If parameter absolute is set to _false_, the URI will *not* be absolutized, it will be returned in its stored form (which could be absolute).

You would probably rather use the uri read-only property, unless you want to set the parameter _absolute_.
