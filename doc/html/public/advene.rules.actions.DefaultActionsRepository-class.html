Caption(self, context, parameters)

Display a message as a caption for a given duration.

If the 'duration' parameter is not defined, a default duration will be used.

DisplayMarker(self, context, parameters)

Display a marker on the video.

If the 'duration' parameter is not defined, a default duration will be used. Parameters: Shape: square, circle, triangle. Color: named color. Position: x, y in percentage of the screen. (0,0) is on top-left. Duration: in ms
