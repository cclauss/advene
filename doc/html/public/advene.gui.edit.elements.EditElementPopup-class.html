Abstract class for editing Advene elements.

To create a specialized edit window, define the make_widget method, which returns the appropriate widget composed of EditAttributesForm and EditTextForm. The make_widget must register its forms via self.register_form().

On validation, the registered forms are asked to update the values of their respective elements.
