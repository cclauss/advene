build_optionmenu(elements, current, on_change_element, editable=True)

Build an ComboBox.

elements is a dict holding (key, values) where the values will be used as labels current is the current activated element (i.e. one of the keys) on_change_element is the method which will be called upon option modification.

Its signature is:

``def on_change_element([self,] element):``

generate_list_model(elements, controller=None, active_element=None)

Update a TreeModel matching the elements list.
