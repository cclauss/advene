Class Query

Simple Query component.

This query component returns a set of data matching a condition from a given source. If the source is not a list, it will return a boolean.

The 'condition' and 'return value' TALES expression will be evaluated in a context where the loop value is stored in the 'element' global. In other words, use 'element' as the root of condition and return value expressions.
