numericTime(self, s)

Converts a time string into a long value.

This function is inspired from the numdate.py example script from the egenix mxDateTime package.

If the input string s is a valid time expression of the form hh:mm:ss.sss or hh:mm:ss or hh:mm, return the corresponding value in milliseconds (float), else None
