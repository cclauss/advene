Class Action

The Action class.

The associated method should have the following signature:

``def method (context, parameters)``

where:
  • context is a advene.tal.AdveneContext holding various information
  • parameters is a dictionary with named parameters, whose values are coded in TALES syntax (and should be evaluated through context).
