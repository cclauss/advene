__init__(self, here, options={})
(Constructor)

Creates a tales.AdveneContext object, having a global symbol 'here' with value 'here' and a global symbol 'options' where all the key- value pairs of parameter 'options' are copied. Of course, it also has all the standard TALES global symbols.

evaluateValue(self, expr)

Returns the object matching the TALES expression expr applied on the given context. If context is an instance of tales.AdveneContext, it will be used directly. If it is another instance, a new AdveneContext will be created with this instance as global symbol 'here'.

interpret(self, view_source, mimetype, stream=None)

Interpret the TAL template available through the stream view_source, with the mime-type mimetype, and print the result to the stream "stream". The stream is returned. If stream is not given or None, a StringIO will be created and returned.
