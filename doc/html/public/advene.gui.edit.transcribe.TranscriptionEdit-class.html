parse_transcription(self, show_ignored=False, strip_blank=True)

Parse the transcription text.

If show_ignored, then generate a 'ignored' key for ignored texts.

If strip_blank, then strip leading and trailing whitespace and newline for each annotation.

Return : a iterator on a dict with keys 'begin', 'end', 'content' (compatible with advene.util.importer)
